## Backend

-   Ruby on Rails 7.0.3
-   PostgreSQL
-   Puma (web server)
-   Devise (authentication)
-   Friendly ID (URL slugs)
-   Pagy (pagination)
-   Active Storage (file storage)
-   Action Text (rich text content)
-   Image Processing

## Frontend

-   Turbo Rails (Hotwire)
-   Stimulus (JavaScript framework)
-   Bootstrap 5 (CSS framework)
-   Bootstrap Icons
-   Trix (rich text editor)
-   Sass (CSS preprocessor)
-   ESBuild (JavaScript bundler)

## Testing

-   RSpec (testing framework)
-   Capybara (system testing)
-   Selenium WebDriver
